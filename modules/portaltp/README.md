# Módulo Portaltp

Este módulo contém os dados e a lógica para interagir com os portais de transparência da empresa Portaltp.

## Estrutura de Arquivos

- `__init__.py`: Arquivo que marca o diretório como um módulo Python.
- `agents.md`: Documentação específica para a coleta de dados da Portaltp.
- `api_metodos/`: Contém a lista de métodos da API e outras informações relevantes.
- `prefeituras.csv`: Lista de prefeituras que utilizam a plataforma da Portaltp.
- `tests/`: Diretório para os testes específicos deste módulo.
